# Global Weather App

A modern weather application built with React and Vite.

## Features

- Current Weather
- Hourly Forecast
- 5-Day Forecast
- Air Quality Index
- Interactive Map
- Voice Search
- Favorite Cities
- Dark Mode
- Weather Charts
- Sunrise & Sunset
- AI Weather Assistant
- PWA Support
- Responsive Design

## Tech Stack

- React
- Vite
- Axios
- OpenWeather API
- Leaflet
- Chart.js

## Installation

npm install

npm run dev

## Build

npm run build

## Preview

npm run preview
